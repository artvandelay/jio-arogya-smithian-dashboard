About This Analysis

This site applies Adam Smith's insight — that durable systems run on self-interest, not goodwill — to Jio Arogya's health operating system. Each dashboard examines the architecture from a different angle: the risk register maps every structural joint; the incentive matrix tests whether each actor's rational self-interest aligns with the system design; the tensions explorer lets you degrade any lever and see the cascade.

Sourced from the final shared bundle (22 Apr 2026), the Strategic Memo, Data Strategy, Principles & Decisions, and speaking notes.
