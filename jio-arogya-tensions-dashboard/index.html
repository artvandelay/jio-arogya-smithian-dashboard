How to Read This

Start with the full architecture. Then move one lever at a time, or select a preset scenario above. Each row shows how the selected configuration changes an operating layer.

  • Green — the layer continues to compound.
  • Amber — the layer still functions but loses strategic advantage.
  • Red — the mechanism that makes the model economic or defensible breaks.

Each lever shows three concrete choices, from full architecture to constrained versions. Click any operating layer or consequence card for detail.
